FREQUENTLY ASKED QUESTIONS?

1) What is Speech Emotion Recognition System?

 

                    SER (Speech Emotion Recognition) is a system that identifies emotional states in speech data 
                    and entered text using voice analysis, text analysis and emotion recognition techniques.

2) How does SER work?

 

                    SER extracts audio attributes by processing audio files and texts. 
                    These attributes are sent to an emotion recognition model trained with machine learning
                    algorithms (for example, SVM, BERT, and ANN). The model determines the emotional states 
                    of the audio data and produces an output.

3. What emotional states can be recognized?

 

                    The SER system can generally recognize basic emotional states, such as happy, 
                    sad, frustration, angry, excited, neutral.

4. What data types are available for the SER system?

 

                    The SER system can use audio data from various sources. These may include conversations, 
                     phone calls, radio broadcasts, or any audio recording,but these files must be in the
                     .wav extension.

5. What machine learning techniques does the SER system use?

 

                         The SER system usually uses machine learning algorithms such as SVM (Support Vector Machine),
                         ANN (Artificial Neural Network), BERT (Bidirectional Encoder Representations from Transformers) 
                         to recognize emotional states.

6. What does the Voting algorithm do?

 

                            Voting algorithm determines the most accurate result by combining the results of 
        emotion recognition and text analysis used in the SER system. This algorithm weights the outputs of both 
        models and takes a vote to determine the result.

7. In which applications can the SER system be used?

 

            The SER system can be used in many areas such as music analysis, user experience evaluation, 
        social media analysis and marketing research. It can also be used in speech recognition systems and 
        in therapy or health applications based on emotional state analysis.

8. How accurate can the SER system produce?

 

                 The accuracy of the SER system depends on the data set used, the training of the model, and the quality 
                of the implementation. A model trained with a good dataset and properly tuned can achieve high 
                levels of accuracy.The average of the accuracy rate currently used in the SER system is approximately 82%.
